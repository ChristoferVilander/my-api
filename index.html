Christofer API

Detta API tillhandahåller enkel JSON‑baserad information om mig. Det är fritt att använda i egna projekt, demos eller experiment.

Huvud‑endpoint

GET https://christofervilander.se/api/v1

Hämta min information

GET https://christofervilander.se/api/v1/me

Exempel på svar

{
  "message": "Välkommen till API v1",
  "endpoints": ["/api/v1/me"]
}

Exempel i JavaScript

fetch("https://christofervilander.se/api/v1/me")
  .then(res => res.json())
  .then(data => console.log(data));

API:et är under utveckling och kommer att utökas med fler endpoints.
